## Online freight logistics service(在线货运物流服务端)
- Manage freight data online and automatically synchronize the current progress of transportation.
- 在线管理货运数据，自动同步运输当前进度。


## Technology use
- Development language：PHP 7.4
- System framework：Laravel 6.0
- Database: Mysql 5.7
- Cache: Redis
